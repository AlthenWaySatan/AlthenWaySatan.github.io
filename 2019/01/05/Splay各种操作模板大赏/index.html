Splay各种操作模板大赏

发表于 2019-01-05 更新于 2022-02-13 分类于 学习笔记

$\ \ \ \ \ \ \ \,$关于Splay操作的复习笔记：

$\ \ \ \ \ \ \ \,$依然的，只记录模板，不讲原理，关于Splay出门左拐百度。

结构

  1. $\tt root$ ：根；
  2. $\tt cnt$ ：下标大小；
  3. $\tt son[0/1][]$ ：左右儿子；
  4. $\tt sum[]$ ：子树大小；
  5. $\tt val[]$ ：节点权值；
  6. $\tt size[]$ ：节点大小；
  7. $\tt fa[]$ ：节点父亲；
  8. $\tt lazy[]$ ：下传标记。

基本操作

  • 标记上传 ($pushup$)
  • 标记下传 ($pushdown$)，根据情况会不一样
  • 左旋+右旋 ($rotate$)
  • 将$x$旋转到$goal$下 ($splay$)

修改

  • 新建节点 ($newnode$)
  • 清空节点 ($clean$)
  • 建立 ($Build$)
  • 插入($Insert$)
  • 删除节点($delete_pos$)
  • 删除($Delete$)
  • 删除第k($Delete_QueryRank$)
  • 区间修改（翻转）($revse$)

询问

  • 查找某个值的节点编号($QueryRank_pos$)
  • 查找某个排名的节点编号($Rank_pos$)

  • 查找排名第k的值($QueryRank$)

  • 查找某个值的排名($Rank$)

  • 集合内前驱的节点编号($pre_pos$)
  • 集合内后继的节点编号($nex_pos$)
  • 集合内前驱的值($pre$)
  • 集合内后继的值($nex$)
  • 前驱($Numpre$)
  • 后继($Numnex$)
  • 查找某个值的节点编号($find$)，不改变结构
