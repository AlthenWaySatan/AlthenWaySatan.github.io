边双连通分量

$\ \ \ \ \ \ \ \,$对于一个有向图，能互相到达的点在一个连通分量，很多时候一个连通分量的点对答案没有影响，或者可以统一安排，那么我们就用tarjan算法把它们缩在一起。

$\ \ \ \ \ \ \ \,$这样就把有向图转换为了一个DAG，会方便处理很多。
